if 语句的格式： 
>if 要判断的条件:
    条件成立时，要做的事情
    ……

if、else的格式：
>if 要判断的条件:
    条件成立时，要做的事情
    ……
else:
    条件不成立时，要做的事情
    ……
    
if、elif、else的格式：
>if 条件1:
    条件1满足执行的代码
    ……
elif 条件2:
    条件2满足时，执行的代码
    ……
elif 条件3:
    条件3满足时，执行的代码
    ……
else:

### 练习题4

**要求:**

1. 输入一个人的身高\(m\)和体重\(kg\)，根据BMI公式（体重除以身高的平方）计算他的BMI指数。

 * 例如：一個52公斤的人，身高是155cm，则BMI为 :

  `52(kg)/1.552^2(cm)= 21.6`

 *  BMI指数：

 ```
  低于18.5：过轻
  18.5-25：正常
  25-28：过重
  28-32：肥胖
  高于32：严重肥胖
 ```

```
weight = float(input("请输入你的体重（单位：kg）："))
height = float(input("请输入你的身高（单位：m）："))

BIM = weight / height ** 2

print("你的BIM指数是：%.2f" %BIM)

if BIM < 18.5:
    print("过轻")
elif BIM < 25:
    print("正常")
elif BIM < 28:
    print("过重")
elif BIM <= 32:
    print("肥胖")
else:
    print("严重肥胖")


```

elif op == "*":
    print("%d %s %d = %d" % (num1, op, num2, num1 * num2))
