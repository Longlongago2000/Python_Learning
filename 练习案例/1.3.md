 3.设计“过7游戏”的程序, 打印出1-100之间除了含7和7的倍数之外的所有数字。

```
# 过7游戏
# 打印出1-100之间除了含7和7的倍数之外的所有数字
i = 1
while i <= 100:
    if i % 7 == 0:
        i += 1
        continue
    elif i % 10 == 7:
        i += 1
        continue
    print(i , end = "\t")
    i += 1
```
